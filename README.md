Este sitio web es una presentación profesional que muestra:

## 🚀 Características Principales

- **Responsive Design**: Totalmente adaptable a dispositivos móviles y desktop

## 📦 Instalación

### Requisitos Previos
- Node.js (versión 16 o superior)
- npm o yarn

### Pasos

1. **Clonar el repositorio**
   ```bash
   git clone https://github.com/tu-usuario/samuelsena-portfolio.git
   cd samuelsena-portfolio
   ```

2. **Instalar dependencias**
   ```bash
   npm install
   ```

3. **Ejecutar servidor de desarrollo**
   ```bash
   npm run dev
   ```
   El sitio estará disponible en `http://localhost:5173`

4. **Construir para producción**
   ```bash
   npm run build
   ```

5. **Vista previa de producción**
   ```bash
   npm run preview
   ```

## 📝 Scripts Disponibles

- `npm run dev` - Inicia servidor de desarrollo con hot reload
- `npm run build` - Compila TypeScript y genera build optimizado
- `npm run preview` - Visualiza la versión de producción localmente
